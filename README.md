

## Instalação de pacotes

Instale os pacotes dentro da pasta do projeto:

```sh
 npm install
   ```
   
 Dentro do terminal, digite o comando para rodar o projeto:

```sh
 npm run dev
   ```
   
   Acesse o link gerado no terminal:
